- 再生/停止: `Space`
- 反転 (水平、データごと): `Ctrl` + `H`
- 保存: `Ctrl` + `S`
- 元に戻す: `Ctrl` + `Z`
- やり直し: `Ctrl` + `Shift` + `Z` または `Ctrl` + `Y`
- コピー（領域選択後）: `Ctrl` + `C`
- 切り取り（領域選択後）: `Ctrl` + `X`
- 貼り付け: `Ctrl` + `V`
- ムーブツール: `V`
- ペンツール: `B`
- 消しゴムツール: `E`
- バケツツール: `G`
- スポイト: `I`
- 長方形選択: `M`
- 投げ縄選択: `L`
- ブラシサイズ縮小: `[`
- ブラシサイズ拡大: `]`
- 前のフレームへ: `←`
- 次のフレームへ / フレームが末尾なら新規追加: `→`
- 選択範囲があれば解除、なければフレーム削除確認: `Delete`
